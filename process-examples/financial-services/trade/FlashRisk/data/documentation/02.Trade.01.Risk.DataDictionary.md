## Process Input Data

### Core Trade Fields

#### tradeId

- **Type:** String
- **Format:** `TRADE-YYYYMMDD-XXXXXX`
- **Required:** Yes
- **Description:** Unique identifier for the trade being analyzed.
- **Example:** `TRADE-20251014-A1B2C3`

#### instrument

- **Type:** Object
- **Required:** Yes
- **Description:** Details of the financial instrument being traded.
- **Properties:**
  - `instrumentId`: (String) Unique identifier for the instrument (e.g., ISIN, CUSIP).
  - `instrumentType`: (Enum) Type of instrument (e.g., `equity`, `bond`, `derivative`).
  - `ticker`: (String) Ticker symbol of the instrument.

#### quantity

- **Type:** Number
- **Required:** Yes
- **Description:** The quantity of the instrument being traded.

#### price

- **Type:** Number
- **Required:** Yes
- **Description:** The price at which the trade is being executed.

#### currency

- **Type:** String
- **Format:** ISO 4217 currency code
- **Required:** Yes
- **Description:** The currency of the trade.
- **Example:** `USD`, `GBP`, `EUR`

#### traderId

- **Type:** String
- **Required:** Yes
- **Description:** The ID of the trader who executed the trade.

## Task Output Data

### Specify Risk Metrics Output

#### riskMetrics

- **Type:** Array of Objects
- **Required:** Yes
- **Description:** The risk metrics to be calculated.
- **Object Properties:**
  - `metricName`: (Enum) The name of the risk metric (e.g., `VaR`, `Greeks`, `StressTest`).
  - `parameters`: (Object) The parameters for the calculation.

### Provide Market Data Snapshot Output

#### marketDataSnapshot

- **Type:** Object
- **Required:** Yes
- **Description:** A snapshot of the market data at the time of the calculation.
- **Properties:**
  - `timestamp`: (DateTime) The timestamp of the market data snapshot.
  - `prices`: (Object) A map of instrument IDs to their prices.
  - `volatilitySurfaces`: (Object) Volatility data for relevant instruments.

### Aggregate Risk Results Output

#### aggregatedResults

- **Type:** Object
- **Required:** Yes
- **Description:** The aggregated results from the on-premise and cloud risk calculations.
- **Properties:**
  - `onPremResults`: (Object) The results from the on-premise calculations.
  - `cloudResults`: (Object) The results from the cloud calculations.
  - `aggregatedMetrics`: (Object) The final, aggregated risk metrics.

### Review Results Output

#### reviewDecision

- **Type:** Enum
- **Values:** `Acceptable`, `Unacceptable`
- **Required:** Yes
- **Description:** The decision of the risk reviewer.

#### reviewComments

- **Type:** String
- **Required:** No
- **Description:** Any comments from the risk reviewer.

## Reference Data

### Risk Limits

#### varLimit

- **Type:** Number
- **Description:** The Value-at-Risk limit for a given trader or desk.

#### positionLimit

- **Type:** Number
- **Description:** The maximum position size allowed for a given instrument.
